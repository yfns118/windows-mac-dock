## 主要功能一览

ObjectDock 支持两种核心 Dock 类型：

**程序坞（QuickLaunch Dock）**
这是最接近 Mac 风格的那种——透明背景、大图标、鼠标悬停时图标会放大弹起，常用程序一目了然。可以完全替代 Windows 原生任务栏放在屏幕底部，也可以挪到左侧、右侧，甚至顶部。

**标签卡 Dock（Tabbed Dock）**
把文件、文档、网址、程序分门别类收进不同标签，藏在屏幕边缘，要用时点一下就弹出来。特别适合经常要翻文件的人。

其他实用特性：
- 支持同时创建**多个 Dock 栏**，游戏、办公各一条，互不干扰
- 图标动画效果可选：缩放、晃动、闪烁……或者都关掉
- **自动隐藏模式**：不用的时候消失，鼠标靠近时出现
- **幽灵模式**：半透明融入桌面，专注工作不分神
- 可隐藏 Windows 原生任务栏，视觉上更接近 Mac
